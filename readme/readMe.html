How to Animate images on Scroll


Pre-requisites:
1. AOS library from Here
2. Bootstrap 3.4.1 CDN from Here
3. JQuery CDN from Here
4. Ajax CDN from Here
5. Bootstrap 3.4.0 CDN from Here

after getting library from AOS, link the aos.js file and aos.css

Add the following to Script tag in your HTML Body AOS.init({ duration: 1500 });

Now right data-aos="fade-right" in img tag to animate the image which fades from right, look for animation styles from AOS Official Site

To delay an animation write data-aos-delay="300" in img tag

To ease animation use data-aos-easing="ease-in-back" in img Tag, There are various ease animation settings gothrough AOS GitHub



Tooltip

For Tooltip use bootstrap default tolltips by writing data-toggle="tooltip" ,
for placement of tooltip write data-placement="auto bottom" and give the title title="Hey this is my Tooltip..!"
